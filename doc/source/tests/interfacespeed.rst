interfacespeed
--------------

Returns the operational interface speed

.. literalinclude:: ../../../examples/interfacespeed.yml


Parameter
~~~~~~~~~

Interface: Interface name. Example `GigabitEthernet1`

Return
~~~~~~

The operational interface speed in Mbit/s

.. code:: json

 {
     "result": 1000,
     "resulttype": "single"
 }

Supported platforms
~~~~~~~~~~~~~~~~~~~

- Network devices with NAPALM

Requirements
~~~~~~~~~~~~

no special requirements
