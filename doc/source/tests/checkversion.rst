checkversion
------------

Returns the OS version of the network device

.. literalinclude:: ../../../examples/version.yml


Parameter
~~~~~~~~~

None

Return
~~~~~~

The OS version string

.. code:: json

 {
     "result": "CSR1000V Software (X86_64_LINUX_IOSD-UNIVERSALK9-M), Version 15.5(2)S, RELEASE SOFTWARE (fc3)",
     "resulttype": "single"
 }

Supported platforms
~~~~~~~~~~~~~~~~~~~

- Network devices with NAPALM

Requirements
~~~~~~~~~~~~

no special requirements
